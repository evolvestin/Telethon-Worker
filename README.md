# Telethon-Worker

Проект представляет собой многопоточный сервис ретрансляции (relay) для Telegram, построенный на базе библиотеки Telethon. Система обеспечивает взаимодействие с целевыми Telegram-ботами через интерфейс управляющего бота, используя промежуточные пользовательские аккаунты (юзерботы).

Ключевые особенности архитектуры:
* **Асинхронная обработка:** Использование asyncio для высокопроизводительного I/O.
* **Двусторонняя коммуникация:** Прозрачная пересылка текстовых сообщений и медиа.
* **Маппинг интерфейсов:** Поддержка и адаптация Inline-клавиатур (ReplyInlineMarkup) с сохранением состояний в SQLite для корректной обработки Callback-запросов.
* **Мультиаккаунтинг:** Возможность одновременного запуска и управления множеством сессий в отдельных потоках.
* **Облачная синхронизация:** Интеграция с Google Drive для автоматического развертывания и подтягивания файлов сессий (.session).
